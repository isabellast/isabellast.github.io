EECS 183 | Elementary Programming Concepts

  1. Scone Shopping List Maker | C++

    2. A dynamic shopping list generator for a fixed scone recipe. Shopping ingredient list amount for shopping list changes based on the amount of scones entered by the user.

  2. Ohhi | C++

    3. A console version of the logic game Ohhi with two binary tile types depicted as X and O. All rows and columns must have equal number of X and O's and no identical columns and rows can be present.

  3. Battleship | C++

    4. A console battleship game with CPU gameplay and file reading functionality. The user is able to choose two game play modes, CPU vs. Human mode and Human (Player as CPU) vs. Human mode.

  4. Elevators | C++

    5. A console game elevators with two playing modes, CPU emulation and human game play. The goal is to get each generated person to the correct floor in the building without exceding the individual's anger level. Each person is generated on a specific floor of the building and the user's input is parsed and executed according to the possible moves within the game.

EECS 281 | Data Structures and Algorithms

  1. Puzzle Solver | C++

    2. A puzzle solver using a backtracking algorithm with two modes, stack and queue. Returns the solution to a given map under the constaints of the the actual map rules, if one is found. If no solution is found the program returns the reachable areas on a given map.

  2. Stock Stimulator | C++

    3. A Stock stimulation program with a time traveler feature. The program processes orders as they are received. Buying and selling between traders happen based off of the program's critiera for trades. The trades are output as they take place and the numer of trade orders processed are output at the end of the trades on file. If time traveler mode is invoked, it calculates which trades would have created the most profit for each trader and reports this at the end of the program.

  3. Pairing Heap | C++

    4. A implimentation of a self-adjusting pairing heap as discussed within the Fredman and Tarjan paper on pairing heaps.

  4. SQL Implimentation | C++

    5. An SQL imitation project, with implimentations of CREATE, INSERT INTO, DELETE FROM, GENERATE INDEX, JOIN, REMOVE, and QUIT functions.

  5. MST Drone Path Finder | C++

    6. A console program which uses Prim's algorithm to find the minimum spanning tree between delivery points on campus assigned to a specific drone. Drone's must pass through border points to go from medical campus to non-medical campus. the minimum distance is returned.

  6. Traveling Sales Person (Heuristic and Optimal)| C++

    7. A traveling sales person project with a Heuristic that returns the semi-optimal path and uses the semi-optimal path as a bound to find the optimal answer through branch and bound algorithm.
